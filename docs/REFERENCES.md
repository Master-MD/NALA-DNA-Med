# References and Inspiration

Status: 2026-05-27

These links are inspiration and implementation references, not medical validation.

## Autonomous Research Loop

- [karpathy/autoresearch](https://github.com/karpathy/autoresearch)
  - Reference idea: bounded AI-agent experimentation where an agent modifies a small training setup, runs fixed-time experiments, and keeps or discards changes based on a metric.
  - NALA-DNA-Med use: possible future workflow for software experiments, retrieval experiments, and model-pipeline tuning, with human approval and audit logs.

## Local AI Memory

- [MemPalace/mempalace](https://github.com/mempalace/mempalace)
  - Reference idea: local-first AI memory with verbatim storage, structured retrieval, ChromaDB/SQLite-style local components, and MCP tooling.
  - NALA-DNA-Med use: possible future local memory layer for project discussions, research decisions, source notes, and audit-friendly recall.

## LLM-Maintained Wiki

- [karpathy/llm-wiki gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)
  - Reference idea: raw sources remain immutable while an LLM maintains a persistent markdown wiki with index, log, entity pages, concept pages, and contradiction checks.
  - NALA-DNA-Med use: planned public and private wiki model for source-backed biomedical research notes.
