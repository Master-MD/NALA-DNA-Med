# NALA-DNA-Med (Monorepo)

This repository contains the source code for the NALA-DNA-Med application across multiple platforms.

## Structure

*   `macOS/`: The native SwiftUI macOS app concept for local-first biomedical project work.
*   `Windows/`: The native WinUI 3 (Windows App SDK) client utilizing Fluent Design and the same LLM-Fit recommendations.

## Core Philosophy

> **CRITICAL BOUNDARY:** NALA-DNA-Med is not medical advice, not a medical device, and not validated for diagnosis, treatment, prescription, or clinical decision-making. 

Both apps are designed as local-first ("CAVEMAN install") solutions to ensure data privacy (TenantShield) and feature dynamic LLM-Fit logic for offline AI capabilities.
