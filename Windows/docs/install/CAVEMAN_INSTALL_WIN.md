# CAVEMAN Install Guide (Windows)

This guide is for non-technical users who want to run NALA-DNA-Med in "Local Lite" mode on Windows 10/11.

## Prerequisites
- A modern Windows 10/11 PC.
- No development tools required.

## Installation Steps
1. Download the `NALA-DNA-Med-Setup.exe` (or `.msixbundle`) from the release page.
2. Double click the installer.
3. Follow the standard Windows installation prompts.
4. Once installed, launch "NALA-DNA-Med" from your Start Menu.

## First Run
1. **Safety Notice**: Read and acknowledge the safety notice. The software is for research only.
2. **System Check**: The app will automatically check your RAM and disk space.
3. **LLM-Fit**: Select the recommended profile (e.g. `Kleiner`) to ensure the AI runs smoothly on your hardware.

You are now ready to use the Dashboard, Projects Vault, and BioLab Demo!
