# Developer Guide

## Build

```bash
swift build
```

## Test

```bash
swift test
```

## Run App Bundle

```bash
./script/build_and_run.sh --verify
```

## Create DMG

```bash
./script/make_dmg.sh
```

## Notes

Local Lite does not require Docker or OrbStack. Core Mode is represented in the UI and documentation, but the v0.1 app keeps heavy services and model downloads opt-in.
