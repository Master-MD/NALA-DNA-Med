# NALA-DNA-Med Fundamentals

This folder preserves the original foundation material for the NALA-DNA-Med project.

## Contents

- `pre-app-ideas/`
  - Early concept packages and mockup bundles.
- `pre-app-pitch/`
  - Pitch deck, pitch PDF, source/data PDF, and source package.
- `pre-screenshots/`
  - Early UI screens for login/TenantShield, projects, upload center, jobs/audit, settings/runtime, and BioLab graph view.
- `Master-MD-NALA-DNA-Med.webloc`
  - Local link artifact for the GitHub repository.

## How to Use This Material

Use these files as historical and conceptual source material. They are not clinical evidence and they are not validation data.

When turning this material into public claims, keep this distinction clear:

- project vision and motivation
- software architecture and UI direction
- research-support workflow ideas
- actual biomedical evidence, which must be cited separately from scientific literature
